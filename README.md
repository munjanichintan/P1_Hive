# P1_Hive

## Project Description

This project aims to analyse movies data by using Apache Hive.

## Technologies Used

* Hortonworks Data Paltform - version 2.6.5
* Apache Hive - version 2.1.1

## Features

* movies of specific genre.

* low rated movies. etc
